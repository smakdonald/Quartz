
### Functions
#### Definitions:

Functions: A *function* is a relationship between two variables for which a unique value of the output variable can be determined from a value of the input variable. In other words, a function is relation in which every input corresponds to exactly one output.

The Vertical Line Test: A graph represents a function if and only if every vertical line intersects the graph in at most one point.
##### Outcomes
- Determine when a relation is a function given a table or equation
- Determine when a relation is a function given a graph using the vertical line test
- Identify the input and output of a function
- Evaluate functions from equations, graphs, and tables
- Identify specific inputs of a function given a corresponding output
