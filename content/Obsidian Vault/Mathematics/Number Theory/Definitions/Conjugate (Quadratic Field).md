Given a [[Quadratic Field|quadratic field]] $Q[\sqrt{d}]$, if $\a = a + b \sqrt{d} \in\Q[\sqrt{d}]$, then the *conjugate* of $\a$ is 
$$\a = a + b \sqrt{d} = a - b \sqrt{d},$$
and its *norm* is given by $$N (\a ) = \a \a = (a + b \sqrt{d})(a - b \sqrt{d}) = a^{2} - db^{2}.$$
